thornboo`s blog

这里是 thornboo 博客的源码站，本项目是基于  [quartz-blog](https://github.com/iceprosurface/quartz-blog) 模板进行修改的。


# 使用说明

1. Get Started  
```bash
git clone https://github.com/jackyzha0/quartz.git
cd quartz
npm i
npx quartz create
```

2. Writing Content  
创作你的博客。

3. Build and Preview  
```bash
npx quartz build --serve
```
